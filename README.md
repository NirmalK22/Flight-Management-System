# Flight Management System

## Description
This project is a Flight Management System developed using Java Swing for the user interface and MySQL as the database backend. It allows users to manage flights, bookings, and passenger information efficiently. The system provides functionalities to add, update, delete, and view flight records, offering a simple and interactive desktop application for airline management.

## Technologies Used
- Java Swing  
- MySQL  
- JDBC  

## Features
- Add, update, and delete flight records  
- Manage passenger bookings  
- Search and view flight details
